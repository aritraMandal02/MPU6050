Connections for our Arduino UNO and NANO are as following:

VCC - 5v

GND - GND

SCL - A5

SDA - A4

INT - D2
===================================================================================================
Upload the MPU6050_DMP6_eg.ino code to your Arduino UNO or Arduino Nano board. Note the COM Port.
Download MPU6050 library and include this library to your Arduino IDE.
Download processing from this link : https://processing.org/download
Download the processing libraries and paste them to '...\processing-4.0b2\modes\java\libraries' folder.
Open the MPUTeapot processing code. In line 74, change the COM Port to your COM port. 
Run the processing code.
Test your MPU6050.
